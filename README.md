# Mythology

Interactive web experience about ancient athletics and mythology.

## Stack

- **React 19** + **TanStack Start / Router**
- **Three.js** via `@react-three/fiber` — 3D scene
- **Tailwind CSS v4** + **shadcn/ui**
- **Framer Motion** + **GSAP** + **Lenis** — animations & smooth scroll
- **Vite** + **Bun**

## Getting started

```bash
bun install
bun run dev
```

Build for production:

```bash
bun run build
bun run preview
```
